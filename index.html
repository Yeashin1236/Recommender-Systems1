Usability Tips

Hosting: This project is client-side and requires the data files (u.data, u.item) to be accessible via the relative path data/. For GitHub Pages, ensure a data folder exists.

Expected Training Time: Training is CPU/GPU intensive. With 10 epochs and WebGL acceleration, it may take 30 seconds to a few minutes depending on your hardware.

PCA Visualization: The scatter plot displays a 2D projection of 1000 sampled item embeddings. Hover over points to see the movie title.
